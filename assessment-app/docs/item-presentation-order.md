# LeapYear Assessment — Item Presentation Order (V2)

**85 items, sequenced for optimal student experience.**

Design principles:
- Opens with CEI-II items (engaging, self-reflective warm-up)
- ORVIS items distributed evenly across scales (never cluster one scale)
- IPIP-NEO items interspersed throughout
- NCS-6 items placed in the middle third
- ORVIS Production items (A31–A33) spread across middle section
- Dweck growth mindset items close the assessment (reflective, philosophical)
- No instrument names or section headers visible to student
- Response scale changes are noted — the UI must switch scale labels at these transitions

## Response Scale Summary (for UI implementation)

The assessment uses 4 different response scales. The UI should display the appropriate scale labels based on the current item's instrument:

| Scale ID | Label | Anchors | Items Using |
|----------|-------|---------|-------------|
| CEI | "How accurately does this reflect you?" | 1 = Very Slightly or Not at All, 2 = A Little, 3 = Moderately, 4 = Quite a Bit, 5 = Extremely | B1–B10 |
| ORVIS | "How much would you enjoy this?" | 1 = Very Much Dislike, 2 = Dislike, 3 = Neutral, 4 = Enjoy, 5 = Very Much Enjoy | A1–A33 |
| IPIP | "How well does this describe you?" | 1 = Very Inaccurate, 2 = Moderately Inaccurate, 3 = Neither, 4 = Moderately Accurate, 5 = Very Accurate | D1–D28 |
| Dweck | "How much do you agree?" | 1 = Strongly Agree, 2 = Agree, 3 = Mostly Agree, 4 = Mostly Disagree, 5 = Disagree, 6 = Strongly Disagree | E1–E8 |

**Note:** The Dweck scale has 6 points; all others have 5. The UI must handle this transition at position 78.
